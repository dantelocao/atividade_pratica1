Abaixo estão as tags HTML utilizadas nas páginas index.html e html_dict.html com suas respectivas descrições:

Tag Descrição
<!DOCTYPE html> Declara o tipo de documento e a versão do HTML.
<html> A raiz de um documento HTML.
<head> Contém metadados sobre o documento, como título e links para estilos.
<title> Define o título da página, mostrado na aba do navegador.
<meta> Define metadados sobre o documento, como charset e viewport.
<body> Contém o conteúdo visível da página.
<h1> - <h6> Define cabeçalhos, com <h1> sendo o mais importante e <h6> o menos importante.
<p> Define um parágrafo.
<a> Define um link.
<address> Fornece informações de contato ou localização.
<nav> Define uma seção de navegação que contém links para outras partes do site.
<ul> Representa uma lista não ordenada, onde os itens são apresentados como marcadore
<li> Denota um item dentro de uma lista, seja ordenada ou não.
<aside> Indica conteúdo relacionado ao conteúdo principal, geralmente usado para barras laterais ou informações adicionais
<header> Representa a parte superior de uma página ou seção, contendo elementos como títulos e menus de navegação.
<footer> Define o rodapé de uma página ou seção, geralmente incluindo informações de copyright e links adicionais.
<main> Marca o conteúdo principal de um documento, que é diretamente relacionado ao tema ou propósito da página.
<script> Utilizada para incluir ou referenciar código JavaScript que pode ser executado no navegador.
